# Dispatch Entry Form with Google Sheets

## Setup Instructions

1. Create a Google Service Account from Google Cloud Console.
2. Share your Google Sheet (name it "Dispatch Data") with the service account email.
3. Download your `credentials.json` and place it in the same folder.
4. Run the app:
   ```bash
   streamlit run app.py
   ```

## Requirements

Install using:

```bash
pip install -r requirements.txt
```
